# Local AI PDF Agent

A **local AI agent** that reads PDF documents and answers questions using a local LLM.

## Features

* AI document assistant
* Semantic search across PDFs
* Summarization
* Quiz generation
* Research reports
* Runs locally (no cloud API)

## Tech Stack

* LangChain
* Ollama
* FAISS
* Streamlit

## Installation

Clone the repository

```
git clone https://github.com/YOURUSERNAME/local-ai-pdf-agent.git
cd local-ai-pdf-agent
```

Install dependencies

```
pip install -r requirements.txt
```

Run the AI agent

```
streamlit run app.py
```

## Requirements

Install Ollama and pull the model:

```
ollama pull llama3.2:3b
```
